Add Google article schema to all Posts

== Description ==

This plugin no need any configulation. Will automatic add script type='application/ld+json' to all posts page.

== Installation ==

### Installation

1. Upload 'top-story-schema' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Done
